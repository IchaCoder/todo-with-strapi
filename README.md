# Todo app made with Strapi CMS

This react todo app was bootstraped by vite.

**Functionalities Involved**

* Login
* Signup
* Add item
* Delete item
* Edit items
* Check completed item

## Dependencies

1. React Router DOM
2. Formik & Yup
3. TailwindCSS
4. [Strapi CMS](https://strapi.io)
5. React icons


_How to start this project locally_


```bash
git clone <repo link>
cd todo-with-strapi
npm install
npm start
```
